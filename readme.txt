Modern WordPress starter theme with Full Site Editing, Gutenberg blocks, accessibility features, and advanced blog functionality.

== Description ==

Zenstarter is a modern, performance-optimized WordPress theme designed for developers and content creators. Built with Full Site Editing (FSE) support, accessibility in mind, and advanced blogging features.

**Key Features:**

* Full Site Editing (FSE) compatible
* Accessibility-ready (WCAG 2.1 AA compliant)
* Advanced blog functionality with intelligent related posts
* Custom Gutenberg blocks and patterns
* Performance optimized with lazy loading and critical CSS
* Translation ready with complete i18n support
* WooCommerce compatible
* Mobile-first responsive design
* SEO optimized with structured data
* Developer-friendly with hook system

**Perfect for:**

* Bloggers and content creators
* Small businesses and portfolios
* E-commerce sites with WooCommerce
* Developers building custom sites
* Agencies and freelancers

== Installation ==

1. In your admin panel, go to Appearance > Themes and click the Add New button.
2. Click Upload Theme and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= Does this theme support Full Site Editing? =

Yes! Zenstarter is built specifically for Full Site Editing with comprehensive theme.json configuration.

= Is the theme accessibility ready? =

Absolutely. Zenstarter follows WCAG 2.1 AA guidelines and includes proper ARIA labels, keyboard navigation, and screen reader support.

= Can I use this theme with WooCommerce? =

Yes, Zenstarter includes full WooCommerce compatibility with custom templates and styling.

= How do I customize the blog features? =

The theme includes extensive hook system and filters. Check the documentation in the docs/ folder for detailed customization examples.

= Is the theme translation ready? =

Yes, all strings are translatable and the theme includes support for RTL languages.
